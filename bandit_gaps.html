Beta Bandit Gaps

Ian Osband

November 5, 2015

I want to do some simple simulations to gain some insights into Beta-Bernoulli bandits. The key to the argument for \(K\)-armed bandits comes from Theorem 3.5 in this paper.

After \(T\) steps with \(K\) arms, at least one arm has been pulled less that \(T/K\) times, and for this arm it is impossible to differentiate between a Bernoulli \(\delta\) and \(\delta + \sqrt{K / T}\). I will now seek to show that, if we have \(K\) arms, each IID with suitable Beta prior we should be able to get this as a pretty typical outcome.

I will generate a bandit problem from the prior \(Beta(\lambda, \lambda)\) with \(\lambda = \sqrt{T / K}\) so that the typical standard deviation of arms is as required. My claim is that a significant portion of arms will be separated from the maximum arm by some reasonable constant to the maximum.

gapScalings <- function(nArms, horizon){
  # gapScalings returns the gaps of the gaps of the bandit problem.
  # Here we only deal with alpha=beta --> mean=0.5
  #
  # Args:
  #   nArms - int - number of arms K
  #   horizon - int - timesteps T
  #
  # Returns:
  #   gapScales - nArms x 1 - gap / (sqrt(K / T))
  lambda <- sqrt(horizon / nArms)
  means <- rbeta(n=nArms, shape1=lambda, shape2=lambda)
  gaps <- max(means) - sort(means)
  return(gaps * lambda )
}

genScaling <- function(nArms, hMul, nSim=100) {
  # genScaling will generate several simulations of scalings.
  
  horizon <- nArms * hMul
  x <- gapScalings(nArms, horizon)
  for (i in 1:nSim) {
    xNew <- gapScalings(nArms, horizon)
    x <- c(x, xNew)
  }
  dt <- data.table(x=x, nArms=nArms, hMul=hMul, horizon=horizon)
  return(dt)
}

armList <- c(10, 100, 1000, 10000)
horizonMul <- c(10, 100, 1000)


dt <- data.table()
for (k in armList) {
  for (L in horizonMul) {
#      print(paste0('arms: ', k, ' hMul: ', L))
    newDt <- genScaling(nArms=k, hMul=L, nSim=min(1000, 1e6/k))
    dt <- rbind(dt, newDt)
  }
}

plt <- (ggplot(dt)
        + aes(x=x)
        + geom_density()
        + facet_grid(nArms ~ hMul, labeller=label_both))
plt

This faceted plot shows the distribution of gaps below optimal. This gap is normalized by the standard deviation \(\sqrt{K / T}\) which is the gap that we ned for our proof. The facets of the graph are by the number of arms and the horizon multiple hMul, which is the factor by which \(T\) is bigger than \(K\).

As you can see, for large values of \(T\) relative to \(K\) we can clearly see that the majority of the arms are some significant scaling away from the maximum gap. Moreover, this scaling is something similar to the standard deviation \(\sqrt{K / T}\).
